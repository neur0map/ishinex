# ishinex

A GUI application and toolkit for Claude Code.

## Development

```bash
# Install dependencies
bun install

# Development mode
bun run tauri dev

# Build production
bun run tauri build
```

## Requirements

- [Claude Code CLI](https://claude.ai/code)
- [Rust](https://rustup.rs/)
- [Bun](https://bun.sh/)
